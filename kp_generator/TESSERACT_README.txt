Установка Tesseract OCR (для распознавания текста в PDF-сканах)
================================================================

Программа использует Tesseract для OCR, если PDF содержит только изображения (скан/фото).

Варианты размещения tesseract.exe:
  1. Папка kp_generator/tesseract/tesseract.exe (рядом с программой)
  2. Файл kp_generator/tesseract.exe (в корне папки с программой)

Скачать Tesseract для Windows:
  https://github.com/UB-Mannheim/tesseract/wiki
  Установите в любую папку и скопируйте tesseract.exe в папку программы
  либо добавьте путь к Tesseract в PATH системы.

Язык по умолчанию: rus (русский). При установке Tesseract выберите данные для русского языка.
